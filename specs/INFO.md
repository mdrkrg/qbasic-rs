# Information about the specs

`./Minimal_Basic_or_QBasic.md` is the implementation specs, modified to include local image paths.

`./original/` contains the original copies of the specs.

- `Minimal Basic (or QBasic)_1763869571482.md`:
  The original specs, containing the requirement of compiling BASIC `LET` statements to Y86-64 assembly (the educational ISA introduced by CSAPP).
- `Minimal Basic (or QBasic)_1764063071801.md`:
  The revised specs. The compilation has become a bonus part, after some controversies and arguments.

`./references/` contains materials referenced in the specs.
